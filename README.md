# Poetry Flow (诗歌流)

基于 Tinder 卡片滑动交互的现代诗歌展示网页应用。精选三首现代诗歌，提供日光与夜幕两种极简配色主题，支持卡片滑动记录情感足迹，以及一键生成包含二维码与项目链接的高清分享海报。
